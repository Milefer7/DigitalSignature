# DigitalSignature

* **DigitalSignature** 是一个提供数字签名功能学习性demo。Go 用于处理后端服务，通过简单的前端展示和交互实现数字签名功能。

## 样式展示

## 如何使用

* 开启后端

  * `go run main.go`

    > windows系统直接双击`DigitalSignature.exe`即可

* 开启前端

  * 打开根目录下`index.html`

* 开启完成。
